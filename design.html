Design

WIREFRAMES

I designed these wireframes for both website and mobile screen so that it would be easier to implement when it's time to make the website responsive. I have three pages done so far, the Home,Design and Blog pages. I started with these because these are the ones I have worked on so far. I will continue with the other frames when they are compulsory.

I did 2 sections. One is the low perosity wireframe which includes basic shapes and shades of grey to indicate where everything will go. The other one includes the deatails of the page and how it will look. The light grey squares indicate the header and the footer which is where nav bars and other information will go. The darker shades will indicate information. The darkest shade or black, will show the blog post images.

Websites and inspiration

I drew inspiration from the apple website with how simplistic it is and how straightforward the UI and navigation are.
